# 英语精听复读播放器

基于尚雯婕英语学习法的专业精听复读工具，支持逐句精听和影子跟读。

## 功能特点

- 🎯 逐句精听复读，支持SRT/LRC字幕文件
- 🎵 支持多种音视频格式（MP4、MP3、AVI、MKV等）
- 🔄 自定义复读次数和间隔时间
- 📝 字幕同步显示，字体大小可调
- 💾 学习进度自动保存
- 📋 播放列表管理

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python english_listening_player.py
```

或直接运行：

```bash
run.bat
```

## 使用方法

1. **添加文件**：点击"播放列表"→"添加文件到播放列表"
2. **自动匹配字幕**：程序会自动查找同目录下的字幕文件
3. **开始学习**：选择文件后点击"播放"开始逐句精听
4. **控制播放**：使用"上一句"/"下一句"按钮控制学习进度
5. **个性化设置**：点击"软件设置"调整字体和复读参数

## 系统要求

- Windows 7/8/10/11 (64位)
- Python 3.7+
- VLC播放器（用于媒体播放）

## 安装VLC

访问 https://www.videolan.org/vlc/ 下载并安装VLC播放器
