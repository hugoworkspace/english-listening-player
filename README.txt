英语精听复读播放器 安装说明

系统要求：
- Windows 7/8/10/11 (64位)
- 至少4GB内存
- 支持DirectX 9.0c的显卡

安装步骤：
1. 双击运行 EnglishListeningPlayer_Setup.exe
2. 按照安装向导提示完成安装
3. 安装完成后可以从开始菜单启动程序

使用方法：
1. 启动程序后，点击"选择视频文件"加载视频
2. 点击"选择字幕文件"加载对应的SRT字幕文件
3. 使用播放控制按钮进行逐句播放和复读

功能特点：
- 逐句精听播放
- 自定义复读次数和间隔
- 字幕同步显示
- 播放进度记忆
- 字体大小调节
